--------------------------------------------------

# CORE DESIGN PRINCIPLES

## 1. Controlled Motion
Animations must feel intentional.

NO:
- random floating elements everywhere
- excessive bounce
- distracting transitions
- chaotic motion

YES:
- smooth inertia
- elegant easing
- subtle parallax
- directional movement
- cinematic reveals

Motion should guide attention.

Every animation must have purpose.

--------------------------------------------------

## 2. Cinematic Pacing

The website should unfold like a movie.

Scrolling must feel:
- immersive
- dramatic
- paced
- emotional

Sections should breathe.

Use:
- large whitespace
- delayed reveals
- slow transitions
- layered storytelling

Avoid:
- information overload
- fast aggressive transitions
- clutter

The user should FEEL the product before reading about it.

--------------------------------------------------

## 3. Minimal UI

UI should disappear into the experience.

Use:
- clean typography
- minimal navigation
- subtle borders
- soft gradients
- glassmorphism carefully
- clean spacing

Avoid:
- too many buttons
- unnecessary cards
- overcrowded sections
- excessive text blocks

Every section should focus on ONE message.

--------------------------------------------------

## 4. Focus Direction

The website must control user attention.

Use:
- light
- contrast
- motion
- typography scale
- depth
- blur
- spacing

to direct the eye.

Important content should feel naturally discoverable.

Never allow the user to feel visually lost.
